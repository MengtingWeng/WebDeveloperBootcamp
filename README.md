# WebDeveloperBootcamp
 The exercise for the The Web Developer Bootcamp
 
sublime 快捷键
```
crtl + s	保存
ctrl + f	浏览器打开文件
ctrl + =	放大
ctrl + -	缩小
ctrl + p	goto anything, 可以搜索某个文档，达到某一行
ctrl + enter	直接进行下一行，加上shift就是进入上一行
ctrl + /	注释，加上shift就是去掉注释
ctrl + ]	往后缩进
ctrl + [	向前缩进
ctrl + z	撤销
ctrl + y	重复
ctrl + j	合并行
ctrl + g	跳转行
ctrl + w	关闭打钱打开文件，加上shift就是关闭所有
ctrl + d	选中所有光标所在处单词，重复按D跳传下一处
ctrl + l	选中所在行，重复按L添加行数
ctrl + k + b	打开侧栏
ctrl + n	新建窗口
ctrl + `	打开console


ctrl + shift + d	复制当前行
ctrl + shift + k	删除当前行
ctrl + shift + m	选中花括号中内容
ctrl + shift + p	调出对应的terminal
ctrl + shift + [	fold
ctrl + shift + ]	unfold
ctrl + shift + l	打散行
ctrl + shift +↑↓	当前行于上面或者下面调整
ctrl + shift + F	查找替换
ctrl + shift + p	package control

shift + alt + 1	单个窗口
shift + alt + 2	窗口分为两列
shift + alt + 3	窗口分为三列
shift + alt + 4	窗口分为四列
shift + alt + 5	窗口分为田字格
shift + alt + 8	窗口分为两行
shift + alt + 9	窗口分为三行
F11	全屏或者退出全屏
